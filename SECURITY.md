# Security Policy

## Supported Versions

We are currently supporting security updates for the following versions of **Paldo ALM**:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in **Paldo ALM**, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

### How to Report

1.  Please report the vulnerability via email to **[jj6side865@gmail.com]**.
2.  In your email, please include:
    * The specific component or module (e.g., `ZenithBrain`, `AegisEngine`).
    * Steps to reproduce the vulnerability.
    * Any relevant code snippets or logs.

### Our Response

* We will acknowledge receipt of your report within 48 hours.
* We will investigate the issue and determine the severity.
* We will work to provide a fix as soon as possible.
* We ask that you maintain confidentiality until we have resolved the issue.

Thank you for helping keep Paldo ALM safe!
